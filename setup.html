Android — Private DNS

Works on every Android phone, regardless of brand or skin.

  1. 1
    Open Settings

    Tap Network & internet → Private DNS. On Samsung it's under Connections → More connection settings. Can't find it? Search Private DNS in Settings.

  2. 2
    Choose "Private DNS provider hostname"

    Some phones label it "Designated Private DNS".

  3. 3
    Paste the hostname below, then tap Save

    That's it — no app, no profile to download.

Keep it from being switched off

Anyone can change Private DNS in Settings. To lock it down, pick one:

Set up a Guest profileHand the child a restricted user that can't reach DNS settings.
Install a third-party App LockerLock the Settings app behind a PIN — plenty of free options on the Play Store.

iPhone & iPad — Configuration profile

A one-tap profile sets secure DNS for the whole device.

  1. 1
    Download the profile below

    Open this page in Safari — Chrome and other browsers can't install profiles.

  2. 2
    Open Settings — tap "Profile Downloaded"

    It appears near the top, just under your name.

  3. 3
    Tap Install and enter the passcode

    Confirm once more and you're protected.

Lock it with Screen Lock

So the profile can't simply be removed again:

Turn on Screen TimeSet a Screen Time passcode, then block changes under Content & Privacy Restrictions → the profile can't be deleted without it.

Mac — Configuration profile

The same profile, installed from System Settings.

  1. 1
    Download the profile below
  2. 2
    Open System Settings → General → Device Management

    The downloaded profile waits there for review.

  3. 3
    Select the Bastion profile and click Install

    Enter the Mac's password to confirm.

Configuration profile
For a shared family Mac, set up the child as a Standard (non-admin) user so the profile can't be removed.

Windows — Encrypted DNS (DoH)

Set secure DNS in Windows 11 or 10 network settings.

  1. 1
    Open Settings → Network & internet → DNS server assignment

    Click Edit next to your active Wi-Fi or Ethernet connection.

  2. 2
    Set to Manual, turn on IPv4

    Set DNS encryption to Encrypted only (DNS over HTTPS).

  3. 3
    Paste the DoH template below, then Save
